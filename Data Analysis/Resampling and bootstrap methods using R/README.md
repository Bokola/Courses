## Resampling and bootstrap methods using R 

This course is focused on Resampling and bootstrap methods using R.  

The course structure:

* Resampling from population.
 * The basic bootstrap.
 * Bootstrap confidence intervals.
 * Bootstrap tests.
 * Linear models.
  * GLMs
